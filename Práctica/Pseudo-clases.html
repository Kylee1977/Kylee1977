Si queremos cambiar el aspecto de un enlace, pero sin distinguir que el enlace se haya visitado o no, tenemos entonces tres opciones:

  • Definir simultaneamente las dos Pseudo-Clases

    • Este parrafo contiene un enlace al parrafo siguiente

    Este parrafo es el destino del enlace anterior

  • Definir únicamente los enlaces no visitados

    • Este parrafo contiene un enlace al parrafo siguiente

    Este parrafo es el destino del enlace anterior

  • Definir los enlaces en general (a [] )
